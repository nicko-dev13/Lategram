# Lategram

Instagram Clone

## SETUP

## For installation on server side

## in root

```
npm install
```

## For installation on client side

cd to client

```
npm install
```

## For the backend Server

```
npm run server
```

## For running the react Server

cd .. (root of the app)

```
npm run client
```

## For running the Node Server as well as React Server

```
npm run dev
```
